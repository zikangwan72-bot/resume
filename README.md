# 万秭康个人简历网站

这是一个使用 React 和 Vite 构建的个人简历网站，计划发布到 GitHub Pages。

## 本地运行

```powershell
npm.cmd run dev
```

## 构建

```powershell
npm.cmd run build
```

## 发布

项目已配置 GitHub Pages 的 Vite 路径：

```text
/wan-zikang-resume/
```

等 Git 工具可用后，将代码推送到 GitHub 仓库 `wan-zikang-resume`，再使用 GitHub Actions 发布。

如果之后要替换真实头像，可以把照片放到 `src/assets/profile.jpg`，再在页面中引用。
